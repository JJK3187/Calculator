## 계산기 만들기
### Step1. 하나의 클래스로 계산기 만들기
<details><summary>
</summary>

하나의 클래스로 기본적인 사칙연산이 가능한 계산기를 만들었다.
while 문을 통해서 반복적으로 계산이 가능하게 구현했으며, try-catch 문을 통해 숫자가 아닌 문자를 입력하는 예외 상황을 처리했다. 연산자가 "/"일때 2번째 정수가 0일 경우 0으로 나눌 수 없는 예외처리를 if문으로 진행했으며 switch 문으로 각 연산자 별로 계산을 따로 진행하게 구현했다.
계산이 끝난 후 n,N을 입력시 계산기가 종료하도록 만들었다.
</details>

### Step2. 계산용 클래스를 구분해서 계산기 만들기

<details><summary>
</summary>

Step1 에서 구현한 기능들 중 계산하는 기능을 따로 Calculator 클래스로 만들었다. Step1 의 계산기능에 더해 계산 결과를 저장하는 기능을 추가했으며 저장된 계산 결과는 private로 접근하지 못하게 작성했다. getter를 통해 저장된 데이터를 불러오는 기능과 setter를 통해 저장된 데이터를 삭제하는 기능을 추가했다.
</details>
